Setup Bookmarklet

Add the "MR ChromaKey" button to your browser

Copy the Bookmarklet Code

Tap the button below to copy the bookmarklet JavaScript to your clipboard.

Create a New Bookmark

In the Quest browser, tap the menu (three dots) and select "Bookmark this page" or go to your bookmarks and create a new one.

Edit the Bookmark

Open your bookmarks, find the one you just created, and edit it. Replace the URL with the code you copied in Step 1. Name it "MR ChromaKey".

On Quest: Open bookmarks → Long press the bookmark → Edit → Replace the URL field with the copied code.

Use It!

Now browse to any website with a green screen video. Make sure the video is playing, then tap your "MR ChromaKey" bookmark. It will detect the video and open the MR ChromaKey app automatically.

Works best with sites that serve direct .mp4 or .webm files (Pexels, Pixabay, stock footage sites). YouTube uses DRM streaming, so the bookmarklet will pass the page URL instead — you may need to paste a direct video link manually.
