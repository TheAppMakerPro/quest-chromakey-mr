Setup Bookmarklet

Add the "MR ChromaKey" button to your browser

Copy the Bookmarklet Code

Tap the button below to copy the bookmarklet JavaScript to your clipboard.

Create a New Bookmark

In the Quest browser, tap the menu (three dots) and select "Bookmark this page" or go to your bookmarks and create a new one.

Edit the Bookmark

Open your bookmarks, find the one you just created, and edit it. Replace the URL with the code you copied in Step 1. Name it "MR ChromaKey".

On Quest: Open bookmarks → Long press the bookmark → Edit → Replace the URL field with the copied code.

Use It!

Browse to any website with green screen videos. Before playing the video, tap your "MR ChromaKey" bookmark. A floating green button will appear on the page. Now play and browse videos normally. When you find one you want, tap the floating button — it grabs the video and opens the app.

The floating button stays on screen even while videos play. Tap it anytime to send the current video to MR ChromaKey. Tap the X to dismiss it. Works best with sites serving direct .mp4/.webm files (Pexels, Pixabay, stock footage sites).
